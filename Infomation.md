# Gitのコマンドのあれこれ

Windowsでは「Git Bash」MACでは「ターミナル」を使ってコマンド操作を行う。

* **●●●のフォルダに移動する**

`cd ●●●`

* **ひとつ上の階層に移動する**

`cd ..`

* **リポジトリを新規に作成する**

`git init`

リポジトリ（repository）貯蔵庫という意味

`git add ●●●`

`git commit -m"■■■"`

`git remote add origin https://▲▲▲`

`git push origin ◇◇◇`

`git pull origin ◇◇◇`

`git clone https://▲▲▲`

`git branch`

`git checkout`
